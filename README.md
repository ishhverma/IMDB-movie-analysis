
# IMDB Movie Analysis: Factors Influencing Movie Success 🎬

# 📋 Project Overview
This project analyzes a comprehensive **IMDb movie dataset** to understand the key factors influencing movie success. The dataset includes information such as **IMDb ratings, budgets, gross earnings, directors, actors, genres, and audience votes**.
The objective of this analysis is to uncover **cinema trends, audience preferences, and industry patterns** by exploring relationships between movie profitability, ratings, genres, and influential contributors in the film industry.

# 🎯 Project Objectives
The main objectives of this analysis were:
* Perform **data cleaning and preprocessing** for accurate insights
* Identify **movies with the highest profit margins**
* Determine **top directors based on average IMDb ratings**
* Analyze **popular movie genres and audience preferences**
* Identify **critic and audience favorite actors**
* Examine **Top 250 movies**, including foreign-language films
* Study **voting trends across different decades**
